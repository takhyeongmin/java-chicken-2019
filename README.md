### 클래스

- OrderedMenus;
  - 아는 것 : 주문한 메뉴들
- OrderedMenu
  - 아는 것 : 메뉴 가격, 메뉴 수량
- Payment

### 기능

- [ ] 주문 등록
  - [x] 테이블의 인덱스 입력
    - [x] 예외 : 테이블의 인덱스 예외처리
  - [x] 메뉴 등록 선택 입력
    - [x] 예외 : 메뉴 등록의 존재 여부
  - [ ] 수량 입력 
    - [ ] 예외 : 합이 99개 이상인지 체크
  - [ ] 테이블 목록에 원 표시 추가하기
- [ ] 결제 하기
  - [ ] 결제  테이블 인덱스 입력받기 
    - [ ] 예외 :테이블의 인덱스 예외 처리
  - [ ] 주문 내역 출력하기
  - [ ] 결제 방식 입력받기
    - [ ] 예외 : 1,2 다른 입력 들어올 경우
  - [ ] 할인율 적용하기
    - [ ] 치킨 10마리당 10000원
    - [ ] 현금 5% 할인 (치킨 금액에서 한번더)
