### 기능

- [x] 주문 등록
  - [x] 테이블의 인덱스 입력
    - [x] 예외 : 테이블의 인덱스 예외처리
  - [x] 메뉴 등록 선택 입력
    - [x] 예외 : 메뉴 등록의 존재 여부
  - [x] 수량 입력 
    - [ ] 예외 : 합이 99개 이상인지 체크
  - [x] 받은 입력들로 OrderMenu 생성
  - [x] 테이블 목록에 원 표시 추가하기
- [x] 기능 입력
- [x] 결제 하기
  - [x] 테이블 선택 입력
  - [x] 테이블 정보 생성
  - [x] 총합 생성하기
  - [x] 결제  테이블 인덱스 입력받기 
    - [x] 예외 :테이블의 인덱스 예외 처리
  - [x] 할인율 적용하기
    - [x] 치킨 10마리당 10000원
    - [x] 치킨의 총량 가져오기
    - [x] 현금 5% 할인 (치킨 금액에서 한번더)
